1. Install git desktop app
2. Create a project in system
3. Create a github account
4. Create a git project after login to github
5. Go to project folder and run the below commands
	> git init
	> git remote add origin https://github.com/saikatmahapatra/sample-web-app-for-poc.git
	> git remote set-url origin https://github.com/saikatmahapatra/sample-web-app-for-poc.git
	> touch .gitignore (craete a .gitignore for excluding directories)
	> git add . (add all untracked files)
	> git commit -m 'first commit'
	> git push origin master
